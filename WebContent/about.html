Welcome to MuV - My Movie

As part of our assignments in the course of Cloud Computing, we created our own version of a video streaming and sharing application and host it on the cloud. Further, we implemented a basic rating system for the videos and display videos in order of these ratings.

Here's a few details about what the functionalities the web application provides:

LIST OF VIDEOS:
The main lists all available videos from the Amazon S3 bucket.

UPLOAD:
When user chooses to upload a new video, it is stored using an Amazon S3 bucket. If a file other than compatible video files are uploaded, it throws an error message. There is an option to delete the videos.

VIDEO STREAMING:
The users can watch the videos on the browser. The videos stored in Amazon S3 bucket are distributed using CloudFront, which is the content distribution server provided by Amazon.

RATING:
The Amazon RDS database is used to store details about the video: name, uploaded timestamp and a cumulative rating (ie, they should be updated and averaged everytime a user rates it). These ratings are updated everytime a new user rates it.

Videos are listed in the order of ratings everytime the page is refreshed.

It uses the below Amazon services -
1. S3 bucket - http://aws.amazon.com/s3/
2. EC2 instance that hosts the application - http://aws.amazon.com/ec2/
3. Cloudfront Distribution Services - http://aws.amazon.com/cloudfront/
4. RDS database - http://aws.amazon.com/rds/
